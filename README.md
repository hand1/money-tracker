Money Tracker App 💸
Money Tracker adalah aplikasi pelacak keuangan pribadi yang ringan, aman, dan mudah digunakan. Aplikasi ini membantu Anda mencatat arus kas, mengatur anggaran, dan menganalisis kesehatan keuangan Anda secara intuitif.

✨ Fitur-Fitur Utama
Pencatatan Transaksi: Tambahkan pemasukan dan pengeluaran dengan mudah.

Analisis Keuangan: Visualisasi data menggunakan grafik untuk memantau performa keuangan.

Manajemen Anggaran: Atur batas limit pengeluaran pada tiap kategori.

Fitur Keamanan: Proteksi aplikasi dengan 4-digit PIN.

Manajemen Kategori: Tambahkan, edit, atau hapus kategori sesuai kebutuhan.

Laporan Ekspor: Unduh laporan keuangan dalam format PDF.

Pencarian & Pagination: Cari transaksi dengan cepat dan navigasi data yang rapi.

🚀 Tutorial Instalasi (Akun Pribadi)
Ikuti langkah-langkah berikut untuk menjalankan aplikasi ini di akun Google Anda:

1. Persiapan Google Apps Script
Buka Google Sheets dan buat spreadsheet baru.

Klik menu Extensions (Ekstensi) > Apps Script.

Di editor, hapus kode yang ada dan ganti dengan kode backend Anda (simpan dengan nama code.gs).

Klik tombol + di sebelah file, pilih HTML, beri nama index.html, lalu salin kode frontend Anda ke dalamnya.

2. Deploy Aplikasi
Klik tombol Deploy > New deployment.

Pilih tipe Web app.

Pada "Execute as", pilih Me (Saya).

Pada "Who has access", pilih Anyone (Siapa saja).

Klik Deploy dan izinkan akses (Authorize access).

Salin Web App URL yang muncul.

3. Tips Optimasi URL
Jika URL dari Apps Script terlalu panjang, Anda dapat menggunakan Google Sites untuk menyematkan aplikasi Anda (Embed):

Buka Google Sites.

Buat situs baru dan tambahkan elemen Embed.

Masukkan Web App URL Anda ke dalam kotak yang tersedia.

Publikasikan situs tersebut untuk mendapatkan URL yang lebih singkat dan mudah diingat.

📖 Tutorial Penggunaan
Login: Masukkan PIN (Default: 1234) jika fitur keamanan aktif.

Tambah Transaksi: Klik tombol +, pilih jenis (Pemasukan/Pengeluaran), masukkan nominal, pilih kategori, lalu simpan.

Analisis: Buka tab grafik 📈 untuk melihat laporan bulanan. Gunakan filter bulan/tahun untuk data spesifik.

Budgeting & Pocket: Buka tab target 🎯 atau kantong 💼 untuk mengatur target tabungan dan batas pengeluaran.

Pengaturan: Klik inisial nama Anda di pojok kiri atas untuk mengatur PIN, mengubah nama, atau menambah kategori baru.

# money-tracker
Aplikasi Money Tracker adalah alat bantu finansial pribadi yang dirancang untuk mencatat, mengelola, dan memantau arus kas harian secara sistematis guna membantu pengguna mencapai kontrol keuangan yang lebih baik melalui fitur pemantauan pemasukan, pengeluaran, anggaran, dan laporan analisis performa keuangan.

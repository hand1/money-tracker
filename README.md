# Money Tracker App 💸

Money Tracker adalah aplikasi pelacak keuangan pribadi yang ringan, aman, dan mudah digunakan. Aplikasi ini membantu Anda mencatat arus kas, mengatur anggaran, dan menganalisis kesehatan keuangan Anda secara intuitif.

## ✨ Fitur-Fitur Utama

* **Pencatatan Transaksi:** Tambahkan pemasukan dan pengeluaran dengan mudah.
* **Analisis Keuangan:** Visualisasi data menggunakan grafik untuk memantau performa keuangan.
* **Manajemen Anggaran:** Atur batas limit pengeluaran pada tiap kategori.
* **Fitur Keamanan:** Proteksi aplikasi dengan 4-digit PIN.
* **Manajemen Kategori:** Tambahkan, edit, atau hapus kategori sesuai kebutuhan.
* **Laporan Ekspor:** Unduh laporan keuangan dalam format PDF.
* **Pencarian & Pagination:** Cari transaksi dengan cepat dan navigasi data yang rapi.

## 🚀 Tutorial Instalasi (Akun Pribadi)

Ikuti langkah-langkah berikut untuk menjalankan aplikasi ini di akun Google Anda:

### 1. Persiapan Google Apps Script

1. Buka [Google Sheets](https://sheets.new) dan buat *spreadsheet* baru.
2. Klik menu **Extensions (Ekstensi)** > **Apps Script**.
3. Di editor, hapus kode yang ada dan ganti dengan kode backend Anda (simpan dengan nama `code.gs`).
4. Klik tombol **+** di sebelah file, pilih **HTML**, beri nama `index.html`, lalu salin kode frontend Anda ke dalamnya.

### 2. Deploy Aplikasi

1. Klik tombol **Deploy** > **New deployment**.
2. Pilih tipe **Web app**.
3. Pada "Execute as", pilih **Me (Saya)**.
4. Pada "Who has access", pilih **Anyone (Siapa saja)**.
5. Klik **Deploy** dan izinkan akses (Authorize access).
6. Salin **Web App URL** yang muncul.

### 3. Tips Optimasi URL

Jika URL dari Apps Script terlalu panjang, Anda dapat menggunakan **Google Sites** untuk menyematkan aplikasi Anda (*Embed*):

1. Buka [Google Sites](https://sites.google.com).
2. Buat situs baru dan tambahkan elemen **Embed**.
3. Masukkan **Web App URL** Anda ke dalam kotak yang tersedia.
4. Publikasikan situs tersebut untuk mendapatkan URL yang lebih singkat dan mudah diingat.

## 📖 Tutorial Penggunaan

1. **Login:** Masukkan PIN (Default: `1234`) jika fitur keamanan aktif.
2. **Tambah Transaksi:** Klik tombol `+`, pilih jenis (Pemasukan/Pengeluaran), masukkan nominal, pilih kategori, lalu simpan.
3. **Analisis:** Buka tab grafik `📈` untuk melihat laporan bulanan. Gunakan filter bulan/tahun untuk data spesifik.
4. **Budgeting & Pocket:** Buka tab target `🎯` atau kantong `💼` untuk mengatur target tabungan dan batas pengeluaran.
5. **Pengaturan:** Klik inisial nama Anda di pojok kiri atas untuk mengatur PIN, mengubah nama, atau menambah kategori baru.
